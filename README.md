\# Health Facility Stratification Index (HFSI)

Version: v0.1.0-alpha ("Foundation")



An open-source, configurable framework for health facility stratification using routine administrative health data.



\## Current Version



\*\*v0.1.0-foundation\*\*



This release establishes the conceptual architecture, measurement framework, workbook structure, and development standards for the Health Facility Stratification Index (HFSI). It serves as the baseline reference implementation upon which subsequent versions will be developed.



\## Overview



The Health Facility Stratification Index (HFSI) is a parameter-driven measurement framework designed to stratify primary healthcare facilities using standardized tracer indicators derived from routine administrative health data.



The project is intended to support:



\- Health facility stratification

\- Evidence-based facility prioritization

\- Routine health systems assessment

\- Operational research

\- Decision support for programme managers
